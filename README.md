# Muneeba Jamil — Personal Portfolio

A fully responsive, single-page personal portfolio website built with React, featuring smooth animations, a custom cursor, and a sleek dark aesthetic.

🔗 **Live Demo:** [muneeba-jamil90.github.io](https://github.com/Muneeba-Jamil90)

---

## ✨ Features

- **Custom Cursor** — animated dot + trailing ring with hover effects
- **Typewriter Hero** — cycling role titles with smooth type/delete animation
- **Floating Particles** — ambient background particle system
- **Scroll Reveal Animations** — sections fade in as you scroll
- **Responsive Design** — works across desktop and mobile
- **Dark Theme** — deep dark background with gold, purple, and teal accents
- **Noise Texture Overlay** — subtle film-grain aesthetic
- **Smooth Navigation** — fixed blur navbar with active link underline animations

---

## 🗂️ Sections

| # | Section | Description |
|---|---------|-------------|
| 01 | **Hero** | Name, animated role titles, key stats, CTA buttons |
| 02 | **About** | Bio, personal info card, current status |
| 03 | **Skills** | Categorized tech stack with interactive tags |
| 04 | **Experience** | Timeline of internships and roles |
| 05 | **Projects** | Grid of 8 personal and academic projects |
| 06 | **Education** | Degrees, coursework, certifications |
| 07 | **Contact** | Email, LinkedIn, GitHub links |

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| UI Library | React 18 (via CDN) |
| Transpiler | Babel Standalone |
| Fonts | Playfair Display · DM Sans · JetBrains Mono |
| Styling | Pure CSS with CSS custom properties |
| Animations | CSS keyframes + Intersection Observer API |
| Deployment | GitHub Pages / Static Hosting |

> No build tools required — runs directly in the browser from a single `index.html`.

## 🚀 Getting Started

### View Locally

```bash
git clone https://github.com/Muneeba-Jamil90/<My-Portfolio>.git
cd <repo-name>
# Open index.html in your browser — no server needed
open index.html
```

### Deploy to GitHub Pages

1. Go to your repo **Settings → Pages**
2. Set source to `main` branch, `/ (root)`
3. Save — your site will be live at `https://Muneeba-Jamil90.github.io/<repo-name>/`
